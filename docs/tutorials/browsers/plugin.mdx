Users can customize their browsing experience according to their preferences and needs by installing plugins.

用户可以通过安装插件根据自己的喜好和需求定制浏览体验。

> To prevent potential backdoors, it’s best to choose open-source plugins whenever possible.

> 为了防止潜在的后门，最好选择开源插件。

Trancy – AI Bilingual Subtitles & Web Immersion Translation

AI 双语字幕&网页沉浸翻译 — Trancy 语言学习

Designed for language learning, this tool enhances users’ language skills through immersive translations and bilingual subtitles while watching videos and browsing web pages.

旨在通过沉浸式翻译和双语字幕功能，帮助用户在观看视频和浏览网页时提升语言能力。

BewlyBewly

A browser extension tailored for Bilibili users, aimed at enhancing the user experience and personalizing the homepage.

专为 Bilibili 用户设计的浏览器扩展，旨在提升用户体验并个性化主页。

DuckDuckGo Privacy Essentials

Provides a safer and more convenient online browsing experience by integrating a range of privacy protection features.

通过集成多种隐私保护功能，它使得在线浏览变得更加安全和便捷。

Global Speed: Video Speed Control

Global Speed: 视频速度控制

Offers flexible control over video and audio playback speeds, significantly enhancing the viewing experience and boosting efficiency—especially useful for those who often watch content at 2.5x speed.

灵活控制视频和音频播放速度，显著提升体验，也大大提升了效率，因为我时常使用 2.5 倍速~

MetaMask

A versatile cryptocurrency wallet that not only makes managing digital assets easy but also serves as a gateway to the decentralized web.

一个多功能的加密货币钱包，不仅方便用户管理数字资产，还为他们提供了进入去中心化网络的桥梁。

React Developer Tools

Enhances the debugging and development workflow by providing a wealth of features, making it indispensable for building and optimizing React applications.

通过提供丰富的功能来增强调试和开发流程，使得 React 应用的构建和优化变得更加高效。如果你使用 React 开发，那么这必不可少！

Vimium C - 全键盘操作浏览器

Inspired by the Vim editor, this extension provides users with a highly efficient, all-keyboard browsing experience, allowing navigation, tab management, and common tasks via keyboard shortcuts, reducing the need for a mouse.

旨在为用户提供高效的全键盘操作体验。该插件受到 Vim 编辑器的启发，允许用户通过键盘快捷键进行网页浏览、标签页管理和其他常见操作，从而减少对鼠标的依赖。

Violentmonkey 暴力猴

A powerful tool that offers extensive user script support for users who want to customize and optimize their web experience.

一个强大的工具，为希望定制和优化网页体验的用户提供强大的用户脚本支持。

Simple Translate

简约翻译

An open-source bilingual translation extension designed to provide users with a simple and efficient web translation experience.

一款开源的双语对照翻译扩展，旨在为用户提供简便、高效的网页翻译体验。

Cookie-Editor

Allows users to create, edit, and delete cookies for the current tab efficiently. This tool is ideal for developers conducting web testing or for users who wish to manage cookies related to privacy manually.

允许用户高效地创建、编辑和删除当前标签页的 cookies。该工具非常适合开发人员进行网页测试，或用户手动管理隐私相关的 cookies。
