## Get Started 快速开始

### 1. Download the starter kit 下载入门套件

`git clone https://github.com/curio-research/keystone-starter-kit.git`

### 2. Run Server 运行服务端

```bash
cd server
go mod tidy
go run main.go
```

### 3. Run Client 运行客户端

```bash
cd client
yarn
yarn start
```
